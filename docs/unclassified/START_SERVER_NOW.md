# 🚀 ManaOSサーバー起動手順

**重要**: サーバーを起動するには、**新しいターミナルウィンドウ**で実行してください。

---

## 📋 起動方法（3つの方法から選択）

### 方法1: バッチファイルを使用（最も簡単）

1. **エクスプローラーで** `start_server.bat` をダブルクリック
2. または、**PowerShellで**以下を実行:

```powershell
.\start_server.bat
```

### 方法2: PowerShellスクリプトを使用

**新しいPowerShellウィンドウを開いて**、以下を実行:

```powershell
cd C:\Users\mana4\Desktop\manaos_integrations
.\start_server.ps1
```

### 方法3: Pythonスクリプトを直接実行

**新しいPowerShellまたはコマンドプロンプトウィンドウを開いて**、以下を実行:

```bash
cd C:\Users\mana4\Desktop\manaos_integrations
python start_server_direct.py
```

---

## ✅ 起動確認

サーバーが起動したら、**別のターミナルウィンドウ**で以下を実行して確認してください:

```bash
# ヘルスチェック
curl http://localhost:9500/health

# 詳細状態
curl http://localhost:9500/status

# サービス状態確認
python check_service_status.py
```

サーバーが起動すると、以下のメッセージが表示されます:

```

manaOS拡張フェーズ 統合APIサーバー起動（直接起動版）
============================================================

統合システムをバックグラウンドで初期化中...

  GET  /status - 初期化進捗ステータス

停止するには Ctrl+C を押してください
```

---

## ⚠️ 注意事項

1. **新しいターミナル**: 必ず新しいターミナルウィンドウで起動してください
2. **初期化時間**: 初期化には数分かかる場合があります
3. **停止方法**: 起動したターミナルで `Ctrl+C` を押してください
4. **ポート9500**: ポート9500が使用中の場合は、別のポートを使用するか、使用中のプロセスを終了してください

---

## 🔍 トラブルシューティング

### ポート9500が使用中の場合

```bash
# ポート9500を使用しているプロセスを確認
netstat -ano | findstr :9500

# プロセスを終了（PIDを確認してから）
taskkill /PID <PID番号> /F
```

### エラーが発生した場合

1. エラーメッセージを確認
2. ログファイルを確認（`logs/` ディレクトリ）
3. 依存関係を確認: `pip install -r requirements.txt`

---

**次のステップ**: 新しいターミナルウィンドウで `start_server.bat` または `python start_server_direct.py` を実行してください。
