# Slack会話機能追加完了

## ✅ 追加した機能

### ローカルLLMとの会話機能

Slack IntegrationにローカルLLMとの会話機能を追加しました。

**機能:**
- ✅ 一般的な会話（「こんにちは」「元気？」など）を自動検出
- ✅ ローカルLLM（`local_llm_helper.py`）を使用して会話
- ✅ GPUモードで自動実行（WSL2経由）
- ✅ File Secretaryコマンドと自動判別

---

## 🎯 使い方

### Slackで会話する

**会話の例:**
- `こんにちは`
- `元気？`
- `PythonでHello Worldを出力するコードを教えて`
- `説明して`
- `質問がある`

**File Secretaryコマンドの例:**
- `Inboxどう？`
- `ファイル整理して`
- `最近のファイル見せて`

---

## 🔄 動作フロー

```
1. Slackでメッセージ送信
   ↓
2. Slack Integration受信（ポート5114）
   ↓
3. 会話モードかコマンドモードかを自動判別
   ↓
4a. 会話モード → ローカルLLMで会話
4b. File Secretaryコマンド → File Secretary API
4c. その他 → Unified Orchestrator
   ↓
5. レスポンスをSlackに返信
```

---

## 📊 会話モードの判定

以下の条件で会話モードと判定されます：

1. **会話キーワードを含む:**
   - 「こんにちは」「こんばんは」「おはよう」
   - 「元気」「どう」「教えて」「説明」「質問」
   - 「？」「?」

2. **短いメッセージ:**
   - 10単語未満のメッセージ

3. **コマンドではない:**
   - `/`で始まらないメッセージ

## 🚀 使用例

### 会話例

**ユーザー:** `こんにちは、元気ですか？`

**Bot:** `こんにちは！元気ですよ、ありがとうございます。何かお手伝いできることはありますか？`

### コード生成例

**ユーザー:** `PythonでHello Worldを出力するコードを教えて`

**Bot:** `PythonでHello Worldを出力するコードは以下の通りです：

\`\`\`python
print("Hello World")
\`\`\`

このコードを実行すると、"Hello World"と表示されます。`
