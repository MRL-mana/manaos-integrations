# 🔍 「Server」タブを見つける完全ガイド

## ⚠️ よくある間違い

### ❌ これは「Server」タブではありません
- LM Studioのメニュー（ヘルプ/アプリメニュー）
- バージョン情報、バグレポート、ドキュメントへのリンクが表示されています

---

## ✅ 「Server」タブの正しい場所

「Server」タブは**左サイドバー**にあります。

### ステップ1: メニューを閉じる

1. 現在開いているメニューを閉じる
   - 画面の外側をクリック
   - または、ESCキーを押す

### ステップ2: 左サイドバーを確認

1. **画面の左側**を見てください
2. **縦に並んだアイコン**があるはずです
3. これが「サイドバー」です

### ステップ3: 「Server」アイコンを探す

サイドバーには以下のようなアイコンが並んでいます：

```
┌─┐
│+│ ← 新規追加
├─┤
│💬│ ← Chat（黄色の吹き出し）
├─┤
│🖥️│ ← Terminal/Console
├─┤
│📁│ ← Files/Models
├─┤
│🔍│ ← Search（紫の虫眼鏡）
├─┤
│🖧│ ← Server ← これです！
├─┤
│📥│ ← Download
└─┘
```

---

## 💡 サイドバーが見えない場合

### 方法1: 左端にマウスを合わせる
- 画面の左端にマウスを移動
- サイドバーが表示される場合があります

### 方法2: ウィンドウの左端を確認
- LM Studioウィンドウの左端を確認
- サイドバーが折りたたまれている可能性があります

### 方法3: キーボードショートカット
- `Ctrl + B` でサイドバーの表示/非表示を切り替えられる場合があります
- （バージョンによって異なる場合があります）

---

## 📋 「Server」タブを開いたら

1. 「**Select a model to load**」をクリック
2. モデルを選択（推奨: Qwen2.5-Coder-7B-Instruct）
3. 「**Start Server**」ボタンをクリック
