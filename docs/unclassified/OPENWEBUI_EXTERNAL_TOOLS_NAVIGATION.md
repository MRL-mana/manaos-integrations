# 🔥 OpenWebUI「外部ツール」設定画面への移動方法

## 📋 現在の状況

現在「一般」設定画面が表示されています。「外部ツール」設定画面に移動する必要があります。

## 🔍 移動方法

### 方法1: 左サイドバーから移動（推奨）

1. **左サイドバーの「設定」セクションを確認**
   - 「設定」セクション内に「外部ツール」という項目があるはずです

2. **「外部ツール」をクリック**
   - 左サイドバーの「設定」→「外部ツール」をクリック
   - これで「外部ツール」設定画面に移動できます

### 方法2: URLから直接アクセス

ブラウザのアドレスバーに以下のURLを入力：

```
http://localhost:3001/admin/settings/external-tools
```

または、現在のURLを以下のように変更：

```
現在: http://localhost:3001/admin/settings/general
変更: http://localhost:3001/admin/settings/external-tools
```

## 📋 「外部ツール」設定画面で確認すべきこと

1. **Tool Serverが登録されているか確認**
   - `http://localhost:9503` が登録されているか確認

2. **Tool Serverの接続状態を確認**
   - 「Connected」になっているか確認
   - 接続されていない場合、接続ボタンをクリック

3. **OpenAPI仕様のURLを確認**
   - `http://localhost:9503/openapi.json` が正しく設定されているか確認

4. **エンドポイントパスを確認**
   - エンドポイントパスが正しいか確認（修正済み: /service_status, /check_errors, /generate_image）

5. **Tool Serverを再登録（必要に応じて）**
   - エンドポイントパスを修正したので、Tool Serverを再登録することを推奨
   - 既存の登録を削除して再登録

## 🔥 レミ先輩の推奨

### 優先度1: 「外部ツール」設定画面に移動

1. **左サイドバーの「外部ツール」をクリック**
   - これが最も簡単な方法

2. **または、URLから直接アクセス**
   - `http://localhost:3001/admin/settings/external-tools`

### 優先度2: Tool Serverの登録状況を確認

1. **Tool Serverが登録されているか確認**
2. **接続状態を確認**
3. **必要に応じて再登録**

### 優先度3: 「ツールの選択」ドロップダウンで確認

1. **チャット画面に戻る**
2. **「ツールの選択」ドロップダウンを確認**
3. **ツールが表示されることを確認**
