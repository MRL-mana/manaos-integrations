# LM Studio モデルロードガイド

## 現在の状況

### 問題
- ✅ モデルはダウンロード済み（8個）
- ✗ モデルがロードされていない（使用できない）

### 原因
LM Studioでは、モデルをダウンロードしただけでは使用できません。
**明示的に「Load」する必要があります。**

## 解決方法

### 方法1: LM Studio GUIでロード（推奨）

1. **LM Studioを起動**
2. **「Chat」タブまたは「Local Models」タブを開く**
3. **使用したいモデルを選択**
   - 推奨: `qwen2.5-coder-32b-instruct` または `ggml-org/qwen2.5-coder-14b-instruct`
4. **「Load」ボタンをクリック**
5. **モデルがロードされるまで待つ**（数秒〜数分）

### 方法2: 自動ロード（検討中）

LM Studio APIにはモデルを自動ロードする機能がないため、現在は手動ロードが必要です。

## ロード確認方法

```powershell
python check_lm_studio_loaded.py
```

ロード済みのモデルは「✓ ロード済み・使用可能」と表示されます。

## 推奨設定

### 優先度1: 32Bモデル（最高精度）
- `qwen2.5-coder-32b-instruct`
- サイズ: 約20-25GB
- 精度: 最高

### 優先度2: 14Bモデル（高精度・バランス）
- `ggml-org/qwen2.5-coder-14b-instruct` または `qwen/qwen2.5-coder-14b-instruct`
- サイズ: 約8GB
- 精度: 高精度

## 注意事項

- モデルをロードすると、GPUメモリを使用します
- 複数のモデルを同時にロードすると、メモリが不足する可能性があります
- 使用するモデルを1つだけロードすることを推奨します
