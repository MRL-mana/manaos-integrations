## 🔍 トラブルシューティング

### Webhookが呼ばれない場合

1. **ngrokが起動しているか確認**:
   ```powershell
   # ngrokウィンドウで確認
   # または
   Invoke-RestMethod -Uri "http://localhost:4040/api/tunnels"
   ```

2. **n8nワークフローが有効化されているか確認**:
   - n8n UIでワークフローのトグルスイッチがONになっているか

3. **Webhook URLが正しいか確認**:
   - Browse AIの設定画面でURLを再確認
   - 先頭・末尾にスペースがないか確認

---

### Slack通知が届かない場合

1. **n8nのワークフロー実行履歴を確認**:
   - Slackノードでエラーが発生していないか
   - HTTP Requestノードのステータスを確認

2. **Slack Webhook URLを再確認**:
   - n8nのSlackノード（またはHTTP Requestノード）でURLを確認

---

### データが正しく抽出されない場合

1. **Browse AIのロボット設定を確認**:
   - 要素の選択が正しいか
   - ページが完全に読み込まれているか

2. **ロボットを再実行**:
   - Browse AIで「Run」を再度クリック
