# 🔧 LM Studioサーバー起動トラブルシューティング

## ⚠️ チェックボックスを有効化したがサーバーが起動しない場合

### 確認事項

#### 1. モデルが選択されているか確認

ヘッドレスモードでは、**モデルを事前に選択する必要がある場合があります**。

**確認方法：**
1. LM Studioの「**Chat**」タブまたは「**Search**」タブを開く
2. 上部の「**モデルを選択してください**」ドロップダウンを確認
3. モデルが選択されているか確認
4. 選択されていない場合は、モデルを選択

**推奨モデル：**
- Qwen2.5-Coder-7B-Instruct
- DeepSeek-Coder-6.7B-Instruct

#### 2. LM Studioのエラーメッセージを確認

1. LM Studioのウィンドウを確認
2. エラーメッセージが表示されていないか確認
3. エラーがある場合は、エラーメッセージに従って対処

#### 3. サーバー起動に時間がかかる場合

- チェックボックスを有効化してから、**数分かかる場合があります**
- 特に初回起動時は、モデルの読み込みに時間がかかります
- 少し待ってから再度確認してください

---

## 🔄 代替方法：GUIからサーバーを起動

ヘッドレスモードがうまく動作しない場合は、GUIから直接起動する方法もあります。

### 手順

1. **「Server」タブを開く**
   - 左サイドバーから「Server」タブを探す
   - または、上部のタブメニューから「Server」を選択

2. **モデルを選択**
   - 「Select a model to load」をクリック
   - ダウンロード済みのモデルを選択

3. **「Start Server」をクリック**
   - ボタンをクリックしてサーバーを起動

---

## ✅ 起動確認

サーバーが起動したら、以下で確認できます：

```powershell
.\check_running_status.ps1
```

または、ブラウザで以下にアクセス：
- `http://localhost:1234/v1/models`

---

## 💡 推奨設定

**常時運用する場合：**
1. まずGUIから「Server」タブでサーバーを起動して動作確認
2. 動作確認後、ヘッドレスモードを有効化

**一時的な使用の場合：**
- GUIから「Server」タブで起動する方法が簡単

---

**サーバーが起動するまで、もう少し待ってから確認してください！**
