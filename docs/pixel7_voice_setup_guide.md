# Open WebUI 音声拡張セットアップガイド

## 📱 Pixel 7で音声読み上げを有効化する手順

### ステップ1: Open WebUIの設定を開く

1. Pixel 7で**レミ（Open WebUI）**を開く
2. 左上のメニュー（☰）→ **「設定」**をタップ
3. **「インターフェース」**を選択

---

### ステップ2: カスタムスクリプトを追加

1. **「カスタムJavaScript」**のセクションを見つける
2. 以下のコードをコピー&ペースト：

```javascript
// 👇 このファイルの内容をコピー
openwebui_voice_script.js
```

**または**、PCで以下のファイルを開いてコピー：
```
C:\Users\mana4\Desktop\manaos_integrations\openwebui_voice_script.js
```

3. **保存**をタップ

---

### ステップ3: 動作確認

1. チャット画面に戻る
2. 右下に**🔊 ON**と**⏹️ 停止**ボタンが表示される
3. レミに話しかける
4. **返答が自動で音声になる**

---

## 🎛️ コントロールボタンの使い方

### 🔊 ON / 🔇 OFF
- タップで自動読み上げのON/OFF切り替え
- 緑色=ON、赤色=OFF

### ⏹️ 停止
- 現在の読み上げを即座に停止
- キューに溜まってるメッセージもクリア

---

## ⚙️ 音声設定のカスタマイズ

コード内の`CONFIG`を編集：

```javascript
const CONFIG = {
    autoSpeak: true,  // 自動読み上げ（true/false）
    rate: 1.1,        // 速度（0.5-2.0）速いほど高い
    pitch: 1.0,       // 音程（0.5-2.0）高いほど可愛い
    volume: 1.0,      // 音量（0.0-1.0）
    lang: 'ja-JP'     // 言語
};
```

---

## 🐛 トラブルシューティング

### 音声が出ない
1. Pixel 7の音量を確認
2. Chrome設定 → サイトの設定 → 音声を許可
3. Android設定 → 言語と入力 → テキスト読み上げを確認

### ボタンが表示されない
1. ページをリロード
2. カスタムJSが正しく保存されているか確認
3. Chromeのデベロッパーツールでエラー確認
