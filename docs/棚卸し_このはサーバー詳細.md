## 8. 追加ストレージ解約時の注意

1. **trinity-n8n は停止中** - 再起動すれば n8n は復帰可能（データは Docker volume に）
2. **civitai_models (32G)** - 母艦へ移行 or 削除検討
3. **ai_models_cache (13G)** - キャッシュのため削除可能
4. **cache (17G)** - 要確認の上削除検討
5. **swapfile (24G合計)** - 解約前にルート側へ移動 or 削除
6. **Docker (33G)** - 不要イメージ削除で削減可能
7. **n8n ワークフロー** - 解約前に Web UI からエクスポート推奨

## 9. 最終チェック（母艦移行済み前提）

> 大半は母艦へ移行済みとのこと。追加ストレージ解約前の最終確認結果。

### 移行済みの可能性が高いもの ✅

| 項目 | 確認結果 |
|------|----------|
| credentials.json / token.json | `/root/` に存在せず → **母艦移行済み** |
| .mana_vault | ディレクトリなし → **母艦移行済み** |
| migration_x280 | 存在（X280移行用データ）→ 移行作業済みの可能性 |

### このはに残っているもの（要判断）

| 項目 | 場所 | 備考 |
|------|------|------|
| /root/.env | 285B | 環境変数のみ。母艦の .env と重複なら削除可 |
| /root/manaos_integrations | あり | 同期コード。ManaOS_Sync で母艦と双方向同期 |
| n8n データ | Docker volume (n8n_data, manaos_v3_mana-n8n-data) | ワークフローは Web UI からエクスポート済みか要確認 |
| /mnt/storage500/n8n | 544K | 小容量 |
| generated_images | 470件 | 母艦にコピー済みか要確認 |

### 削除候補（母艦移行済みなら）

| ディレクトリ | サイズ | 削除で解放 |
|--------------|--------|------------|
| civitai_models | 32G | 母艦に同じモデルがあれば |
| ai_models_cache | 13G | キャッシュのため再生成可 |
| cache | 17G | 要確認の上 |
| **合計** | **62G+** | |

### 解約前チェックリスト

- [ ] n8n ワークフローを Web UI (http://100.93.120.33:5679) からエクスポート
- [ ] generated_images で母艦にないものを必要ならコピー
- [ ] /root/.env の必要値を母艦 .env に取り込み済みか確認
