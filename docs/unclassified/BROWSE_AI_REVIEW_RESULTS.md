### Step 3: Webhookを設定（重要）

**n8nと連携するためにWebhookを設定します**:

1. **「統合する」タブをクリック**
2. **「Webhooks」または「Add Integration」をクリック**
3. **Webhook URLを入力**:
   ```
   https://xxxx-xxxx-xxxx.ngrok-free.app/webhook/browse-ai-webhook
   ```
   （ngrokのURLを使用）
4. **「Save」または「Add」をクリック**

---

### Step 4: ngrokでパブリックURLを取得（まだの場合）

**ローカル開発環境からBrowse AIに接続するには**:

1. **ngrokをインストール**（まだの場合）:
   ```powershell
   choco install ngrok
   ```

2. **ngrokでトンネルを作成**:
   ```powershell
   ngrok http 5678
   ```

3. **出力されたURLをコピー**:
   ```
   https://xxxx-xxxx-xxxx.ngrok-free.app/webhook/browse-ai-webhook
   ```
