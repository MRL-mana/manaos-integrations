## 2. メインストレージ (/) 詳細

### 2.1 使用量内訳

| パス | サイズ | 備考 |
|------|--------|------|
| /root | ~11G | ユーザーデータ・プロジェクト |
| /var | ~15G | ログ、キャッシュ、lib |
| /usr | システム | 通常パッケージ |
| /home | 2.5M | ほぼ空 |

### 2.2 /root 主要項目（移行・削除後）

- **snap** 94M - Snapパッケージ
- **manaos_integrations** 24K - X280連携スクリプトのみ（同期コードは別管理）
- **各種DB・設定** - security_audit.db, manaos_unified_metrics.db 等
- **.env** - 環境変数（母艦と重複なら削除可）

### 2.3 /var 内訳

| サブディレクトリ | サイズ | 備考 |
|------------------|--------|------|
| /var/log | ~1.6G | ログ（truncate済み） |
| /var/lib | ~5G | snapd(3.8G), apport, prometheus, apt等 |
| /var/cache | 216M | キャッシュ |

### 2.4 メインで実施済みの削除

- external (google_drive) 1.4G
- logs クリア
- next16-mcp-demo 538M
- trinity_workspace 3.1G（storage500に稼働版あり）
- manaos_command_hub 261M → 母艦移行済み
- manaos-knowledge 161M
- ai_simulator 115M
- 各種古いプロジェクト・一時ファイル

## 3. 追加ストレージ (/mnt/storage500) 詳細

### 3.1 使用量上位

| ディレクトリ | サイズ | 備考 |
|--------------|--------|------|
| docker | 19G | イメージ・overlay2（稼働中コンテナ使用） |
| trinity_workspace | 4.7G | **Trinity稼働用**（古いbackup削除済み） |
| docker-data | 4.7G | Docker データ |

| docker-volumes | 403M | Docker volume |
| mongodb | 157M | MongoDB データ |
| postgres | 70M | PostgreSQL 関連 |
| langfuse-postgres | 46M | Langfuse DB |
| node_exporter | 20M | メトリクス |
| n8n, x280 | - | **削除済み**（母艦へ移行） |

### 3.2 稼働中サービスで使用

- trinity_workspace, openwebui, docker, docker-data, docker-volumes
- mongodb, postgres, langfuse-postgres

### 3.3 追加で実施済みの削除

- x280, n8n → 母艦へ移行後に削除
- civitai_*, cache, ai_models_cache, google-drive, data, python-packages
- offload, system_offload, user_data, logs, gpu_env, binaries
- docker system prune（未使用イメージ・ボリューム）
- 各種空・キャッシュディレクトリ

ssh konoha "du -sh /root/* 2>/dev/null | sort -hr | head -20"
