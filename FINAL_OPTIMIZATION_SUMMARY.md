## 📝 使用方法

### キャッシュ取得
```bash
GET /api/cache/get?key=example_key&type=api_response
```

### キャッシュ保存
```bash
POST /api/cache/set
{
  "key": "example_key",
  "value": {"data": "example"},
  "type": "api_response",
  "ttl_seconds": 3600
}
```

### キャッシュ統計取得
```bash
GET /api/cache/stats
```

### パフォーマンス統計取得
```bash
GET /api/performance/stats
```
