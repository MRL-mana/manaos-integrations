## 📋 利用可能なツール

現在、以下の3つのツールが利用可能です：

1. **service_status**
   - docker/systemdの死活確認
   - Dockerコンテナの状態を確認

2. **check_errors**
   - journalctl/docker logsを要約してエラー検知
   - 最近のログを確認してエラーを検出

3. **generate_image**
   - ComfyUI APIを叩いて画像生成
   - 画像生成してファイルパスを返す

## 🎯 使い方

### 方法1: 自動選択（推奨）

1. **チャットで質問を送信**
   - 例：「dockerコンテナの状態を確認して」
   - 例：「最近のログにエラーがないか確認して」
   - 例：「ComfyUIで画像を生成して」

2. **LLMが適切なツールを自動的に選択**
   - LLMが質問内容に応じて適切なツールを選択
   - ツールが自動的に呼び出される

3. **結果が表示される**
   - ツールの実行結果がチャットに表示される

### 方法2: 手動選択

1. **「ツール5」ポップアップメニューを開く**
   - チャット入力欄の左側のツールアイコンをクリック

2. **使用したいツールを選択**
   - 「manaOS ツールサーバー」を選択
   - 特定のツールを選択（必要に応じて）

3. **チャットで質問を送信**
   - 選択したツールが使用される
