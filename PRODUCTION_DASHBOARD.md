## 🔧 トラブルシューティングガイド

### よくある問題と対応

```
【高レイテンシが発生した場合】
  1. Prometheus ダッシュボード確認
  2. Jaeger で遅いトランザクション追跡
  3. 必要に応じてフェーズを一段階下げる
  4. リソース追加またはキャッシュ調整

【エラー率上昇時】
  1. Loki でエラーログを分析
  2. スタックトレース確認
  3. 関連サービス再起動検討
  4. 必要に応じてロールバック

【リソース不足】
  1. CPU/メモリ詳細を確認
  2. 不要なプロセス停止
  3. スケーリング検討
  4. リソース購入申請
```
