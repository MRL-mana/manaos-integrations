# 🔍 「Server」タブの見つけ方

## 📍 場所

「Server」タブは**左サイドバー**にあります。

### 現在の画面状態
- ✅ 「Chat」タブが開いています（黄色の吹き出しアイコンがハイライト）
- ⚠️  「Server」タブは左サイドバーのアイコンの1つです

---

## 🎯 見つけ方

### ステップ1: 左サイドバーを確認

画面の**左側**にある縦に並んだアイコンを確認してください：

```
┌─┐
│+│ ← 新規追加
├─┤
│💬│ ← Chat（現在選択中・黄色）
├─┤
│🖥️│ ← Terminal/Console（緑）
├─┤
│📁│ ← Files/Models（ピンク）
├─┤
│🔍│ ← Search（紫）
├─┤
│📥│ ← Download
└─┘
```

### ステップ2: 「Server」タブを探す

「Server」タブは通常、以下のようなアイコンで表示されます：
- **サーバー/ネットワークアイコン**（例: サーバーの絵、またはネットワーク接続のアイコン）
- **「Server」という文字**が表示されている場合もあります

**位置**: Chatアイコンの下、またはSearchアイコンの近くにあります

### ステップ3: クリック

「Server」タブのアイコンを**クリック**すると、サーバー設定画面が表示されます。

---

## 💡 見つからない場合

### 方法1: アイコンにマウスをホバー
- 左サイドバーの各アイコンにマウスを合わせると、ツールチップ（説明）が表示されます
- 「Server」と表示されるアイコンを探してください

### 方法2: キーボードショートカット
- 一部のバージョンでは、キーボードショートカットで「Server」タブを開ける場合があります
- 試してみてください: `Ctrl + S` または `Ctrl + Shift + S`

### 方法3: メニューから探す
- 上部のメニューバー（もしあれば）から「View」→「Server」を探す

---

## 📋 「Server」タブを開いたら

1. 「**Select a model to load**」をクリック
2. モデルを選択（推奨: Qwen2.5-Coder-7B-Instruct）
3. 「**Start Server**」ボタンをクリック

---

**左サイドバーのアイコンを確認して、「Server」タブを見つけてください！**
