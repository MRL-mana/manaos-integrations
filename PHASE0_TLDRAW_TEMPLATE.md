## 🎨 描くべき要素

### 1. 中心：ManaOS コア

```
┌─────────────────┐
│   ManaOS Core   │
│                 │
│  - 統合API      │
│  - タスク実行   │
│  - 監視システム │
└─────────────────┘
```

### 2. 周辺：3つの環境

#### 🖥️ 母艦（このはサーバー）
- メインサーバー
- 24時間稼働
- 重い処理

#### 💻 ローカル（新PC）
- 開発環境
- 軽量処理
- マナの作業環境

#### 📱 外部（X280）
- モバイル環境
- リモートアクセス
- バックアップ

### 3. AIエージェント：3姉妹

#### 🤖 レミ（Remi）
- **役割**: 判断・実行
- **場所**: Cursor / 統合API
- **強み**: コード生成、自動化

#### 🌙 ルナ（Luna）
- **役割**: 監視・分析
- **場所**: ダッシュボード / ログ
- **強み**: データ分析、予測

#### ⭐ ミナ（Mina）
- **役割**: 記憶・学習
- **場所**: メモリシステム / RAG
- **強み**: 知識管理、検索

### 4. データフロー

```
情報収集 → 処理 → 保存 → 通知
   ↓         ↓      ↓      ↓
Browse    n8n   Obsidian  Slack
```

### 5. 主要サービス

- **n8n**: 自動化ワークフロー
- **Obsidian**: 思考・メモ
- **Notion**: 実績・記録
- **Slack**: 通知・コミュニケーション
- **RAG**: 知識検索
- **Ollama**: ローカルLLM
- **ComfyUI**: 画像生成

## 🧠 描いた後に確認すること

1. **「今どこが強くて、どこが詰まりやすいか」**
   - 強み: 太い線で強調
   - 詰まり: 赤でマーク

2. **「データはどこから来て、どこへ行くか」**
   - 入力: 青の矢印
   - 出力: 緑の矢印

3. **「何が自動で、何が手動か」**
   - 自動: 緑の箱
   - 手動: 黄色の箱
