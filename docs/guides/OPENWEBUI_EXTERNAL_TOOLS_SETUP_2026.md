# 🔥 OpenWebUI External Tools設定ガイド（2026年最新）

## 📋 重要な発見

OpenWebUIの設定画面で「外部ツール」の設定を確認する必要があります。

## 🔍 確認手順

### Step 1: 「外部ツール」設定画面を開く

1. **OpenWebUIの設定画面を開く**
   - 右上の設定アイコン（⚙️）をクリック
   - または、URLから直接: `http://127.0.0.1:3001/admin/settings/external-tools`

2. **左サイドバーで「外部ツール」を選択**
   - 現在「一般」が選択されている場合、「外部ツール」をクリック
   - 左サイドバーの「設定」→「外部ツール」をクリック

### Step 2: Tool Serverの登録状況を確認

1. **Tool Serverが登録されているか確認**
   - `http://127.0.0.1:9503` が登録されているか確認
   - 登録されている場合、接続状態を確認（「Connected」になっているか）

2. **登録されていない場合**
   - 「Add Tool」または「ツールを追加」をクリック
   - 以下の情報を入力：
     ```
     Name: manaOS Tool Server
     URL: http://127.0.0.1:9503
     OpenAPI Spec: ON（チェックを入れる）
     OpenAPI Spec URL: http://127.0.0.1:9503/openapi.json
     ```
   - 「Save」をクリック

### Step 3: 接続状態を確認

1. **接続状態が「Connected」になっているか確認**
   - 「Connected」になっていない場合、接続ボタンをクリック
   - エラーメッセージがないか確認

2. **接続できない場合**
   - Tool Serverが起動中か確認（http://127.0.0.1:9503/health）
   - OpenAPI仕様が取得できるか確認（http://127.0.0.1:9503/openapi.json）
   - エンドポイントパスが正しいか確認（修正済み: /service_status, /check_errors, /generate_image）

### Step 4: エンドポイントパスの変更に対応

エンドポイントパスを修正したので、以下の点を確認：

1. **Tool Serverを再登録（推奨）**
   - 既存の登録を削除
   - 新しいエンドポイントパスで再登録

2. **または、既存の登録を確認**
   - 既存の登録が正しく動作しているか確認
   - 接続を再試行

### Step 5: チャットコントロールセクションを確認

1. **チャット画面に戻る**
   - 設定画面を閉じて、チャット画面に戻る

2. **「ツールの選択」ドロップダウンを確認**
   - 「ツールの選択」ドロップダウンを開く
   - Tool Serverのツール（service_status, check_errors, generate_image）が表示されることを確認

## 🔥 レミ先輩の推奨

### 優先度1: 「外部ツール」設定画面を確認

1. **左サイドバーで「外部ツール」を選択**
   - これが最重要

2. **Tool Serverの登録状況を確認**
   - 登録されているか確認
   - 接続状態を確認

### 優先度2: Tool Serverを再登録

エンドポイントパスを修正したので、Tool Serverを再登録することを推奨：

1. **既存の登録を削除**
2. **新しいエンドポイントパスで再登録**
3. **接続状態を確認**

### 優先度3: チャットコントロールセクションを確認

1. **「ツールの選択」ドロップダウンを確認**
2. **ツールが表示されることを確認**

## 📋 確認チェックリスト

- [ ] 「外部ツール」設定画面を開いた
- [ ] Tool Serverが登録されている
- [ ] Tool Serverの接続状態が「Connected」
- [ ] OpenAPI仕様のURLが正しく設定されている（http://127.0.0.1:9503/openapi.json）
- [ ] エンドポイントパスが正しい（/service_status, /check_errors, /generate_image）
- [ ] Tool Serverを再登録した（エンドポイントパス変更後）
- [ ] 「ツールの選択」ドロップダウンでツールが表示される
