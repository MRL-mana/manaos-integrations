# S/A/B/破綻フォルダ運用（マナ式・オプション）

## 📁 フォルダ構成

マナ式の「S/A/B/破綻」フォルダ運用に移行する場合の設定です。

```
organized_images/
├── S/          (手元保存級：人間が即OK)
├── A/          (使える)
├── B/          (素材)
└── 破綻/       (捨て or 学習用)
```

## 🎯 分類基準

### スコア閾値（推奨）

| フォルダ | スコア範囲 | 説明 |
|---------|-----------|------|
| **S** | >= 0.85 | 手元保存級（人間が即OK） |
| **A** | 0.75 - 0.85 | 使える |
| **B** | 0.65 - 0.75 | 素材 |
| **破綻** | < 0.65 | 捨て or 学習用 |

### 実装方法

`organize_images_by_evaluation.py`の`FOLDER_STRUCTURE`を以下に変更：

```python
FOLDER_STRUCTURE = {
    "S": {
        "min_score": 0.85,
        "description": "手元保存級（人間が即OK、スコア >= 0.85）"
    },
    "A": {
        "min_score": 0.75,
        "max_score": 0.85,
        "description": "使える（0.75 <= スコア < 0.85）"
    },
    "B": {
        "min_score": 0.65,
        "max_score": 0.75,
        "description": "素材（0.65 <= スコア < 0.75）"
    },
    "破綻": {
        "max_score": 0.65,
        "description": "捨て or 学習用（スコア < 0.65）"
    }
}
```

## 💡 運用のメリット

1. **現場の脳に刺さる**: high/medium/low より直感的
2. **判断が早い**: S/A/B/破綻で即判断
3. **学習データの質向上**: 破綻フォルダから学習パターンを抽出

## 🔄 移行方法

現在の`high_quality/medium_quality/low_quality`から移行する場合：

```python
# 既存の画像を再分類
python organize_images_by_evaluation.py --use-sab-structure
```

## ⚙️ 設定例

### 閾値のカスタマイズ

```python
# より厳しい基準
S: >= 0.90
A: 0.80 - 0.90
B: 0.70 - 0.80
破綻: < 0.70

# より緩い基準
S: >= 0.80
A: 0.70 - 0.80
B: 0.60 - 0.70
破綻: < 0.60
```

## 🎯 まとめ

S/A/B/破綻フォルダ運用は、**現場で使える**分類方法です。

- **S**: そのまま使える
- **A**: 軽微な調整で使える
- **B**: 素材として使える
- **破綻**: 学習データとして活用

この分類により、運用コスパが爆上がりします！
