##  swapfile について

追加ストレージ解約前に、メモリ使用量を下げた状態で以下を実行:

```bash
# 使用量を確認
free -h
swapon --show

# 解約直前に（他プロセスを止めてから）
swapoff /mnt/storage500/swapfile
swapoff /mnt/storage500/swapfile2
rm /mnt/storage500/swapfile /mnt/storage500/swapfile2
```

※ swapoff はメモリに空きがないと失敗します。サービスを止めてから実行してください。
