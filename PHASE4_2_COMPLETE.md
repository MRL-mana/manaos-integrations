## 🔧 実装詳細

### メトリクス収集システム

**エンドポイント**:
- `POST /api/metrics` - メトリクスを記録
- `GET /api/metrics` - メトリクスを取得
- `GET /api/statistics` - 統計情報を取得
- `POST /api/cleanup` - 古いメトリクスを削除

**メトリクスタイプ**:
- `response_time` - レスポンス時間
- `error_rate` - エラー率
- `request_count` - リクエスト数
- `resource_usage` - リソース使用率
- `success_rate` - 成功率

### インテリジェントリトライシステム

**機能**:
- 指数バックオフ（デフォルト）
- サーキットブレーカー
- リトライ可能なエラーの自動判定
- デコレータによる簡単な統合

**使用例**:
```python
from intelligent_retry import retry

@retry(max_retries=3, circuit_breaker_key="api_service")
async def call_api():
    # API呼び出し
    pass
```
