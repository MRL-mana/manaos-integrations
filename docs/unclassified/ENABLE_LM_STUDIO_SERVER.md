# ✅ LM Studioサーバー有効化ガイド

## 🎯 見つけた設定

現在表示されている画面：
- **「Settings」タブ** → **「Developer」サブタブ**
- **「ローカルLLMサービス (ヘッドレス)」**セクション

---

## 📋 サーバーを有効化する手順

### ステップ1: チェックボックスを有効化

1. **「ローカル LLM サービスを有効にする」**のチェックボックスを**クリック**
2. チェックが入ると、サーバーが自動的に起動します

### ステップ2: 確認

チェックボックスを有効にすると：
- LM Studioアプリを開いていなくてもサーバーが動作します（ヘッドレスモード）
- サーバーは `http://localhost:1234` で起動します

---

## ✅ 起動確認

チェックボックスを有効にしたら、以下で確認できます：

```powershell
.\check_running_status.ps1
```

または、ブラウザで以下にアクセス：
- `http://localhost:1234/v1/models`

---

## 🔄 2つの起動方法の違い

### 方法1: 「Server」タブから起動（一時的）
- LM Studioアプリを開いた状態で「Server」タブから起動
- アプリを閉じるとサーバーも停止
- 手動でモデルを選択して起動

### 方法2: 「Settings」→「Developer」で有効化（常時起動）
- ✅ **現在の方法**
- チェックボックスを有効にするだけで自動起動
- LM Studioアプリを閉じてもサーバーが動作し続ける
- ヘッドレスモードで動作

---

## 💡 推奨設定

**常時運用する場合**は、この方法（方法2）がおすすめです：
- ✅ アプリを閉じてもサーバーが動作
- ✅ PC再起動後も自動起動（設定次第）
- ✅ 常時利用可能

---

2. サーバーが起動するまで数秒待つ
3. 起動確認: `.\check_running_status.ps1`
