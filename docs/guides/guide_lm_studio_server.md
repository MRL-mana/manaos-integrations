# 🎯 LM Studioサーバー起動ガイド（詳細手順）

## 📋 ステップバイステップ手順

### ステップ1: LM Studioを前面に表示

1. LM Studioのウィンドウをクリックして前面に表示
2. ウィンドウが最大化されていなくてもOK（見える状態であればOK）

### ステップ2: 「Server」タブを開く

1. LM Studioの**上部**にあるタブメニューを確認
2. 「**Server**」というタブを**クリック**
   - 通常は「Chat」「Search」「Server」などのタブがあります
   - 「Server」タブをクリックすると、サーバー設定画面が表示されます

### ステップ3: モデルを選択

1. 「**Select a model to load**」というドロップダウンまたはボタンを探す
2. クリックしてモデル選択メニューを開く
3. **ダウンロード済みのモデル**を選択
   - 推奨: **Qwen2.5-Coder-7B-Instruct**
   - または: **DeepSeek-Coder-6.7B-Instruct**
   - まだダウンロードしていない場合は、先に「Search」タブでモデルをダウンロード

### ステップ4: サーバーを起動

1. モデルを選択したら、「**Start Server**」ボタンを探す
2. 「**Start Server**」ボタンを**クリック**
3. サーバーが起動すると、以下のような表示が確認できます：
   - `Server running on http://127.0.0.1:1234`
   - `OpenAI Compatible API` が有効になっている
   - ボタンが「Stop Server」に変わる

---

## ✅ 起動確認

サーバーが起動したら、以下で確認できます：

```powershell
# すべてのサービスを確認
.\check_running_status.ps1

# または直接確認
Invoke-WebRequest -Uri "http://127.0.0.1:1234/v1/models" -Method GET
```

**期待される結果：**
- ✅ LM Studioサーバー: 起動中
- ✅ LLMルーティングAPI: 起動中 (`http://127.0.0.1:9502`)
- ✅ 統合APIサーバー: 起動中 (`http://127.0.0.1:9502`)

## 🖼️ 画面の見つけ方

### 「Server」タブの場所
- LM Studioの**上部**にタブメニューがあります
- 「Chat」「Search」「Server」などのタブが横並びになっています
- 「Server」タブをクリックしてください

### 「Select a model to load」の場所
- Serverタブを開くと、画面の上部または中央に表示されます
- ドロップダウンメニューまたはボタンの形式です
- クリックすると、ダウンロード済みのモデル一覧が表示されます

### 「Start Server」ボタンの場所
- モデルを選択した後、画面の下部または右側に表示されます
- 大きなボタンで、「Start Server」と表示されています
- クリックするとサーバーが起動します

---

## 🔧 トラブルシューティング

### 「Server」タブが見つからない場合
- LM Studioのバージョンが古い可能性があります
- 最新版に更新してください: https://lmstudio.ai/download

### モデルが選択できない場合
- モデルがダウンロードされていない可能性があります
- 「Search」タブでモデルを検索してダウンロードしてください

### 「Start Server」ボタンが押せない場合
- モデルが選択されていない可能性があります
- 先にモデルを選択してください

### サーバーが起動しない場合
- ポート1234が他のアプリケーションで使用されていないか確認
- LM Studioを再起動して再試行

---

## 📝 モデルがダウンロードされていない場合

1. 「**Search**」タブをクリック
2. 検索ボックスに「**Qwen2.5-Coder-7B-Instruct**」と入力
3. 検索結果からモデルを選択
4. 「**Download**」ボタンをクリック
5. ダウンロード完了まで待機（数GB〜数十GB、時間がかかります）
6. ダウンロード完了後、Serverタブに戻ってモデルを選択
