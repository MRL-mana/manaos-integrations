# ✅ Settingsからサーバーを有効化する方法

## 📍 現在の画面

- ✅ 「Settings」タブが開いています
- ✅ 「General」サブタブが選択されています

---

## 🎯 サーバーを有効化する手順

### ステップ1: 「Developer」サブタブを開く

1. **「Settings」タブ**が開いていることを確認
2. サブタブメニューを確認：
   - 「General」（現在選択中）
   - 「Chat」
   - **「Developer」** ← これをクリック
   - 「Integrations」

3. **「Developer」サブタブをクリック**

### ステップ2: サーバーを有効化

1. 「Developer」タブを開くと、以下のセクションが表示されます：
   - **「ローカルLLMサービス (ヘッドレス)」**

2. **「ローカル LLM サービスを有効にする」**のチェックボックスを**クリック**
   - チェックが入ると、サーバーが自動的に起動します

3. 説明文：
   - "LM Studio の LLM サーバーを、LM Studio アプリケーションを開いておくことなく使用する"
   - これがヘッドレスモードです

---

## ✅ 起動確認

チェックボックスを有効化したら、以下で確認できます：

```powershell
.\check_running_status.ps1
```

または、ブラウザで以下にアクセス：
- `http://localhost:1234/v1/models`

---

## 💡 注意事項

### モデルが選択されている必要があります

ヘッドレスモードでは、**モデルを事前に選択する必要がある場合があります**。

**確認方法：**
1. 「Chat」タブまたは「Search」タブを開く
2. 上部の「モデルを選択してください」ドロップダウンを確認
3. モデルが選択されているか確認
4. 選択されていない場合は、モデルを選択

---

## 🔄 代替方法：GUIから直接起動

ヘッドレスモードがうまく動作しない場合は、GUIから直接起動する方法もあります。

### 手順

1. **左サイドバーから「Server」タブを探す**
   - Chatアイコンの下にあるサーバー/ネットワークアイコン

2. **「Server」タブをクリック**

3. **モデルを選択**
   - 「Select a model to load」をクリック
   - ダウンロード済みのモデルを選択

4. **「Start Server」をクリック**
   - ボタンをクリックしてサーバーを起動

---

**「Developer」タブを開いて、チェックボックスを有効化してください！**
