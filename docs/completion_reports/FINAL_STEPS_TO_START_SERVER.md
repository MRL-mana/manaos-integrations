- ⚠️  サーバーがまだ起動していません

## 🔍 確認事項

### 1. モデルが選択されているか確認

ヘッドレスモードでは、**モデルを事前に選択する必要があります**。

**確認方法：**
1. LM Studioの「**Chat**」タブを開く
2. 上部の「**モデルを選択してください**」ドロップダウンを確認
3. モデルが選択されているか確認
4. 選択されていない場合は、モデルを選択

**推奨モデル：**
- Qwen2.5-Coder-7B-Instruct
- DeepSeek-Coder-6.7B-Instruct

### 2. サーバー起動に時間がかかる場合

- チェックボックスを有効化してから、**数分かかる場合があります**
- 特に初回起動時は、モデルの読み込みに時間がかかります
- 少し待ってから再度確認してください

### 3. LM Studioのエラーメッセージを確認

1. LM Studioのウィンドウを確認
2. エラーメッセージが表示されていないか確認
3. エラーがある場合は、エラーメッセージに従って対処

---

## 🔄 代替方法：GUIから直接サーバーを起動

ヘッドレスモードがうまく動作しない場合は、GUIから直接起動する方法が確実です。

### 手順

1. **左サイドバーから「Server」タブを探す**
   - Chatアイコンの下にあるサーバー/ネットワークアイコン

2. **「Server」タブをクリック**

3. **モデルを選択**
   - 「Select a model to load」をクリック
   - ダウンロード済みのモデルを選択

4. **「Start Server」をクリック**
   - ボタンをクリックしてサーバーを起動

---

## ✅ 起動確認

サーバーが起動したら、以下で確認できます：

```powershell
.\check_running_status.ps1
```

または、ブラウザで以下にアクセス：
- `http://localhost:1234/v1/models`
