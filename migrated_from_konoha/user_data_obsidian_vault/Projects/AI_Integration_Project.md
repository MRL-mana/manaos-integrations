# AI統合プロジェクト

## プロジェクト概要
ObsidianとNotionの同期システムにAI機能を統合し、自動的なコンテンツ最適化を実現するプロジェクト。

## 技術スタック
- Python 3.10+
- Google AI Studio (Gemini API)
- Notion API
- SQLite (データベース)
- asyncio (非同期処理)

## 実装済み機能
- [x] 基本的な同期システム
- [x] Google AI Studio統合
- [x] Luna Memoエージェント
- [x] Notion Formatterエージェント
- [x] 自動エラーハンドリング
