# 🔍 「Server」タブの場所

## ⚠️ 重要なポイント

**「Server」タブは、設定画面（Settings）の中にはありません。**

---

## 📍 現在表示されているもの

### 「Settings」タブ内のサブタブ
- **General** - 一般設定
- **Chat** - チャット設定
- **Developer** - 開発者設定（「ローカルLLMサービス」の設定がある）
- **Integrations** - 統合設定

これらのサブタブの中に「Server」タブはありません。

---

## 🎯 「Server」タブの正しい場所

「Server」タブは、**メインのナビゲーション**にあります。

### 場所1: 上部のメインナビゲーション

現在表示されている上部のナビゲーションバーを確認：
- Model Search
- Runtime
- Hardware
- Settings（現在選択中）
- mana4

**「Server」タブは、このメインナビゲーションの中にある可能性があります。**

### 場所2: 左サイドバー

画面の**左側**にある縦に並んだアイコンを確認：
- ➕ 新規追加
- 💬 Chat
- 🖥️ Terminal
- 📁 Files
- 🔍 Search
- **🖧 Server** ← これです！
- 📥 Download

---

## 🔄 「Server」タブを見つける方法

### ステップ1: 「Settings」タブから戻る

1. 現在「Settings」タブが開いています
2. メイン画面に戻る方法：
   - 「Settings」タブ以外のタブをクリック（例：「Runtime」や「Hardware」）
   - または、左サイドバーの「Chat」アイコンをクリック

### ステップ2: メイン画面で「Server」タブを探す

1. **上部のメインナビゲーション**を確認
   - 「Server」タブがあるか確認

2. **左サイドバー**を確認
   - 縦に並んだアイコンの中に「Server」アイコンがあるか確認

3. **検索機能を使う**
   - LM Studioの検索機能（Ctrl+F）で「Server」と検索

---

## 💡 見つからない場合

### 方法1: LM Studioのバージョンを確認

- バージョンが古い可能性があります
- 最新版に更新: https://lmstudio.ai/download

### 方法2: LM Studioを再起動

- アプリを完全に閉じて再起動
- 再起動後、「Server」タブが表示される場合があります

### 方法3: キーボードショートカット

- `Ctrl + S` または `Ctrl + Shift + S` を試す
- （バージョンによって異なる場合があります）

---

## 📋 代替方法：ヘッドレスモードを使用

「Server」タブが見つからない場合でも、**ヘッドレスモード**を使用できます：

1. **「Settings」→「Developer」タブ**を開く（既に開いています）
2. **「ローカル LLM サービスを有効にする」**にチェック（既に有効化されています）
3. **モデルを選択**（「Chat」タブでモデルを選択）
4. サーバーが自動的に起動します

ただし、ヘッドレスモードがうまく動作しない場合は、GUIから直接「Server」タブで起動する方法が確実です。

---

**「Settings」タブを閉じて、メイン画面で「Server」タブを探してください！**
