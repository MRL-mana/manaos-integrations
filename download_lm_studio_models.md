## ダウンロード方法

### 方法1: LM Studio GUIから手動ダウンロード

1. **LM Studioを起動**
2. **「Search」タブを開く**
3. **モデル名を検索**
   - `qwen2.5-coder-32b-instruct` を検索
4. **モデルを選択**
5. **「Download」をクリック**
6. **ダウンロード完了を待つ**

### 方法2: コマンドラインから確認

```powershell
# モデル確認
python check_lm_studio_models.py
```

## 注意事項

- 32Bモデルは約20GBの容量が必要です
- ダウンロードには時間がかかります（インターネット速度による）
- 現在の14Bモデルでも十分な精度が得られています

## 推奨設定

現在の設定で十分な精度が得られるため、32Bモデルのダウンロードは**オプション**です。

より高い精度が必要な場合のみ、32Bモデルをダウンロードしてください。
