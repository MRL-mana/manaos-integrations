# GTD Inbox

キャプチャしたすべての「気になること・タスク・アイデア」の一時置き場。

## ルール
- 判断せずにとにかく投げ込む（後でレビューで振り分け）
- ファイル名: `YYYYMMDD_HHMM_<タイトル>.md`
- 週次レビュー（`/review`）で整理

## 振り分け先
- やる（2分以上）→ `../next-actions/items/` or `../projects/items/`
- いつかやる → `../someday/`
- 待ち → `../waiting/`
- 参照ノート → `../../zettelkasten/fleeting/` or `permanent/`
- 不要 → 削除
