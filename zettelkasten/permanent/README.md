# Permanent Notes（永久ノート）

自分の言葉で書いた「知識の原子単位」。

## ルール
- 1ファイル = 1アイデア（atomic note）
- ファイル名: `YYYYMMDD_<slug>.md`
- 末尾に「参照元ノート」「関連ノート」リンクを必ず付ける
- 既存ノートと対話させてから作成（`/zettel` コマンド）

## テンプレート
```markdown
# タイトル

## 本文
（自分の言葉で1アイデアを記述）

## 参照元
- [[literature/...]]

## 関連ノート
- [[YYYYMMDD_...]]

## タグ
#zettelkasten #<topic>
```
