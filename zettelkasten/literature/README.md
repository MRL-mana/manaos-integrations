# Literature Notes（文献ノート）

外部情報源（記事・書籍・会話）の要約・抜き書きノート。

## ルール
- 1ファイル = 1情報源
- ファイル名: `YYYYMMDD_<source-slug>.md`
- 「著者/情報源の主張」を自分の言葉で要約する
- Permanent noteの原材料として使用

## テンプレート
```markdown
# タイトル

- 情報源: （URL / 書籍名 / 会話）
- 取得日: YYYY-MM-DD

## 要点
（箇条書きで主張を整理）

## メモ・コメント
（自分の考察）

## 関連 Permanent Note
- （作成予定または既存）
```
