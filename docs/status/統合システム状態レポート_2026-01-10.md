## 📈 活用の推奨事項

### 1. サーバーの起動
統合システムを活用するには、まずUnified API Serverを起動してください：

```bash
python start_server_direct.py
```

または

```bash
python start_server_with_notification.py
```

### 2. 初期化の確認
サーバー起動後、以下のエンドポイントで状態を確認できます：

- `GET /health` - ヘルスチェック（常に200を返す）
- `GET /ready` - レディネスチェック（初期化完了まで503）
- `GET /status` - 初期化進捗ステータス

### 3. 統合機能の使用
初期化が完了すると、以下の機能が利用可能になります：

- LLMルーティング（`/api/llm/route`）
- 記憶システム（`/api/memory/store`, `/api/memory/recall`）
- 通知送信（`/api/notification/send`）
- 秘書ルーチン（`/api/secretary/*`）
- 画像ストック（`/api/image/*`）

1. **初期化の完了を待つ**: サーバー起動後、初期化が完了するまで数秒かかる場合があります。
