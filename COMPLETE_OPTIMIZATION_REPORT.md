## 🔧 使用方法

### サービスブリッジ

**並列サービスチェック**:
```python
bridge = ManaOSServiceBridge()
services = bridge.check_manaos_services(use_parallel=True)
```

**ワークフロー並列実行**:
```python
workflows = [
    {"type": "image_generation", "params": {...}},
    {"type": "model_search", "params": {...}}
]
results = bridge.execute_workflows_parallel(workflows, max_workers=3)
```

**統合状態とメトリクス取得**:
```python
status = bridge.get_integration_status()
print(status["metrics"])  # メトリクス
print(status["cache_stats"])  # キャッシュ統計
print(status["performance_stats"])  # パフォーマンス統計
```
