## 🎯 モデルロード方式

### 動的モデル管理の仕組み

1. **使用時に自動ロード**
   - ManaOSサービスがモデルを使用する際、自動的にメモリにロード
   - 初回使用時のみロード時間がかかる

2. **未使用時の自動アンロード**
   - 300秒（5分）間未使用のモデルは自動的にアンロード
   - メモリ使用量を最適化

3. **同時ロード制限**
   - 最大2つのモデルを同時にメモリに保持
   - VRAM使用率80%を超えると自動的にアンロード

4. **役割別モデル選択**
   - LLM最適化サービスが役割に応じて最適なモデルを選択
   - 軽量タスクには軽量モデル、重いタスクには大型モデル
