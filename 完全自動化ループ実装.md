### ステップ2: n8nワークフローを作成

#### 2-1. n8nにアクセス

```
http://100.93.120.33:5678
```

#### 2-2. ワークフローをインポート

1. 「Workflows」→「Import from File」
2. `n8n_workflow_template.json`を選択
3. 「Import」をクリック

#### 2-3. 認証情報を設定

各ノードで認証情報を設定:
- Google Drive: Google Drive API認証情報
- Obsidian: Obsidian API認証情報（必要に応じて）
- Slack: Slack API認証情報（必要に応じて）

#### 2-4. Webhook URLを取得

1. Webhookノードを開く
2. 「Listen for Test Event」をクリック
3. Webhook URLをコピー
   ```
   http://100.93.120.33:5678/webhook/comfyui-generated
   ```

#### 2-5. 環境変数に設定

```powershell
$env:N8N_WEBHOOK_URL = "http://100.93.120.33:5678/webhook/comfyui-generated"
```

#### 2-6. ワークフローを有効化

1. ワークフローを開く
2. 右上の「Active」スイッチをON
3. 「Save」をクリック

### ステップ3: 完全自動化ループのテスト

```powershell
# 画像生成を実行
$body = @{
    prompt = "a beautiful landscape, mountains, sunset, highly detailed"
    width = 512
    height = 512
    steps = 20
} | ConvertTo-Json

Invoke-RestMethod -Uri "http://localhost:9500/api/comfyui/generate" `
    -Method POST `
    -Body $body `
    -ContentType "application/json"
```

**確認項目:**
1. ✅ ComfyUIで画像生成が開始される
2. ✅ 生成完了時にn8n Webhookが呼び出される
3. ✅ n8nワークフローが実行される
4. ✅ Google Driveにファイルがアップロードされる
5. ✅ Obsidianにノートが作成される（設定済みの場合）

## 🔧 トラブルシューティング

### n8n Webhookが呼び出されない

1. **環境変数の確認**
   ```powershell
   $env:N8N_WEBHOOK_URL
   ```

2. **統合APIサーバーのログを確認**
   - n8n Webhook通知のログを確認

3. **n8nワークフローが有効か確認**
   - n8nでワークフローが「Active」になっているか確認

### ワークフローが実行されない

1. **Webhook URLが正しいか確認**
2. **n8nの実行履歴を確認**
3. **各ノードの認証情報を確認**
