# 両方の場所へのアクセス方法

## 現在の状況

- **新しいワークスペース**: `C:\Users\mana4\Desktop\manaos_integrations` ✅
- **元のOneDriveフォルダ（母艦）**: `C:\Users\mana4\OneDrive\Desktop\manaos_integrations` ✅

## アクセス方法

### PowerShellから

```powershell
# 新しいワークスペース
cd C:\Users\mana4\Desktop\manaos_integrations

# 元のOneDriveフォルダ（母艦）
cd C:\Users\mana4\OneDrive\Desktop\manaos_integrations
```

### エクスプローラーから

1. **新しいワークスペース**: `C:\Users\mana4\Desktop\manaos_integrations`
2. **元のOneDriveフォルダ**: `C:\Users\mana4\OneDrive\Desktop\manaos_integrations`

### Cursorから

Cursorで両方のフォルダを開くことができます：

1. **File > Open Folder...** で任意のフォルダを開けます
2. 複数のフォルダを同時に開くことも可能です

## 注意事項

⚠️ **両方のフォルダが存在する状態**:
- 新しいワークスペースで作業することを推奨します
- 元のOneDriveフォルダは、Cursorを閉じてから削除することをお勧めします
- 両方で編集すると、どちらが最新か分からなくなる可能性があります

## 推奨アクション

1. ✅ 新しいワークスペース（`C:\Users\mana4\Desktop\manaos_integrations`）で作業
2. ✅ 元のOneDriveフォルダは削除するか、バックアップとして残す
3. ✅ 両方にアクセスする必要がある場合は、読み取り専用で使用
