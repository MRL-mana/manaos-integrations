API エンドポイント

POST /api/v1/images/generate
画像生成リクエスト。job_id を返すのでポーリングで結果取得。
GET /api/v1/images/{job_id}
ジョブステータス確認 (queued → processing → scoring → completed)
GET /api/v1/images/{job_id}/result
完了済みジョブの生成結果（画像URL、品質スコア、コスト）
POST /api/v1/images/enhance-preview
プロンプト強化のプレビュー（実際の生成は行わない）
GET /api/v1/images/dashboard
生成統計、品質スコア分布、RLAnythingダッシュボード
GET /api/v1/images/billing
課金情報、残りクォータ、プラン詳細
GET /metrics
Prometheus テキストフォーマットのメトリクス
