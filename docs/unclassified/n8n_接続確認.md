# n8n接続確認

## このはサーバーのn8n

n8nコンテナは正常に起動しています。以下のURLでアクセスできます：

- **HTTP**: http://100.93.120.33:5678
- **HTTPS**: https://100.93.120.33:5679

### 接続できない場合の確認事項

1. **ブラウザのキャッシュをクリア**
   - `Ctrl + Shift + Delete` でキャッシュをクリア
   - またはシークレットモードで試す

2. **別のブラウザで試す**
   - Chrome、Edge、Firefoxなど

3. **ファイアウォールの確認**
   - ポート5678、5679がブロックされていないか確認

4. **このはサーバーの状態確認**
   ```bash
   ssh konoha
   docker ps | grep n8n
   docker logs trinity-n8n --tail 50
   ```
