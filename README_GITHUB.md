# GitHub活用 - クイックリファレンス

## 🚀 すぐに始める

### 1. プライベートリポジトリを作成

**GitHubで手動作成（推奨）:**
1. https://github.com/new にアクセス
2. リポジトリ名: `manaos-integrations`
3. ⚠️ **必ず「Private」を選択**
4. 「Create repository」をクリック

### 2. リポジトリに接続

```bash
python setup_private_repo.py
```

### 3. 日常的な操作

```bash
python github_workflow.py
```

## 📚 詳細ドキュメント

- `GITHUB_SETUP_STEPS.md` - セットアップ手順
- `GITHUB_PRIVACY_NOTICE.md` - プライバシー設定
- `GITHUB_USAGE_GUIDE.md` - 詳細ガイド
- `GITHUB_ACTIVE_USAGE.md` - 実践的な使い方

## ⚠️ 重要事項

- **必ずプライベートリポジトリとして作成**
- `.env`ファイルは自動的に除外されます
- 認証情報は絶対にコミットしないでください
